# Sesam Landing Page

A modern, empathetic landing page for Sesam - the AI-powered documentation assistant for German psychotherapists.

## Tech Stack

- **React 18** with TypeScript
- **Vite** for fast development and building
- **Tailwind CSS** with custom color palette (Sage Green, Soft Sand, Deep Ocean Blue)
- **Lucide React** for icons
- **React Router** for client-side routing

## Getting Started

### Prerequisites

- Node.js 18+ 
- npm or yarn

### Installation

```bash
# Install dependencies
npm install

# Start development server
npm run dev
```

The site will be available at `http://localhost:5173`

### Build for Production

```bash
npm run build
```

Output will be in the `dist/` directory.

## Deployment on Vercel

1. Push this repo to GitHub
2. Import the repo in Vercel
3. Vercel will auto-detect Vite and configure the build
4. Deploy!

## Design System

### Colors

- **Sage Green** (`sage-*`): Trust & Health - Primary accent color
- **Soft Sand** (`sand-*`): Warmth & Humanity - Secondary warmth
- **Deep Ocean** (`ocean-*`): Professionalism & Security - Text and dark elements
- **Warm backgrounds** (`warm-white`, `warm-gray`): Soft off-white backgrounds

### Typography

Uses "Plus Jakarta Sans" for a clean, legible, and friendly appearance.
